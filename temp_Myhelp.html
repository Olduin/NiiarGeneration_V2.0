1. НАЗНАЧЕНИЕ ПРОГРАММЫ

Приложение выписки заявок для ООО "НИИАР ГЕНЕРАЦИЯ" предназначено для создания заявок на выписку техники и работ с данными заявок.

1.1. Функции программного продукта:

  1. ­ Введение данных, полученных при обращениях клиентов;
  2. ­ Хранение данных заявок;
  3. ­ Создание отчетов на основании составленных заявлений.

1.2. Приложение позволяет хранить данные о:

  1. Технике;
  2. ­ Заявителях;
  3. ­ Видах работ;
  4. ­ Типах заявок.

2. УСЛОВИЕ ПРИМЕНЕНИЯ ПРОГРАММЫ

2.1. Минимальный состав аппаратных средств

  1. ­ ОС: Windows 7 / Windows 8.1 / Windows 10 / Windows 10 Enterprise Long-Term Servicing Branch (LTSB);
  2. ­ ОЗУ: не менее 500 мб;
  3. ­ свободное пространство на жестком диске - не менее 300 мб.

2.2. Минимальный состав программных средств:

MS Word 2007 и выше.

2.3. Требование к персоналу

Пользователь программы (оператор) должен обладать минимальными навыками работы с графическим пользовательским интерфейсом Windows.

3. ВЫПОЛНЕНИЕ ПРОГРАММЫ

3.1. Запуск и загрузка программы

Запустите программу, выведется основная форма "Заявки".

При запуске приложения всплывает титульная форма с информацией о программе. Далее седует форма "Заявки", она является основной формой. Из неё мы переключаемся ко всем остальным модулям.

На ней располагаются элементы:

Кнопка "Создать заявку", при нажатии на которую мы попадаем в форму "строки заявки" для внесения информации о заявках;
  • Кнопка "Справочник техники", открывающая таблицу с зарегистрированной техникой.
  • Кнопка "Справочник типов", открывающая таблицу с внесенными типами заявок.
  • Кнопка "Виды работ", открывающая таблицу с видами работ, имеющимися на предприятии.
  • Кнопка "Заявители", открывающая таблицу со списком заявителей и ответсвенных лиц, который ведётся на предприятии.
  • Таблица "Заявки" со списком имеющихся заявок. При нажатии на заявку из этого списка мы попадаем на форму "Строки заявки", где можем редактировать данную заявку.

Все модальные формы имеют схожие элементы управления и функции. Список с элементами при нажатии на который открывается форма редактирования.

  • Кнопка "Добавить", открывающая форму с пустыми полями элемента;
  • Кнопка "Удалить", удаляет выбранный в таблице элемент;
  • Кнопка "Сохранить", сохраняет изменения;
  • Кнопка "Отмена", Отменяет изменения и возвращает на предыдущую форму;
  • Список элементов, при нажатии на эленмент которого открывается форма редактирования с данными выбранного элемента.
